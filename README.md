# Image-to-Image Translation with Conditional GAN

This project implements a Conditional Generative Adversarial Network (cGAN) for image-to-image translation and conditional image generation, combining insights from multiple research papers and implementations.

## Features

- **Conditional Generation**: Generate images based on class labels
- **Image-to-Image Translation**: Convert images from one domain to another
- **Flexible Architecture**: Supports both conditional generation and pix2pix-style translation
- **Training Monitoring**: Visualize training progress and generated samples
